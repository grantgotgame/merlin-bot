# Skill: Shift Change

*Open a new shift. Close the old one. ~3 minutes.*

---

## Triggers

- "Shift change" / "First shift" / "Second shift"
- "Good morning" / "Let's roll"
- First session of a new shift

## Shifts

| Shift | Hours (flex) | Character |
|-------|-------------|-----------|
| First | ~9am – ~5pm | Meetings, decisions, focus work |
| Second | ~7pm – ~1am | Building, exploration, deep work |
| Night | ~1am – ~9am | Autonomous agents (optional) |

The time between shifts is untracked. That's just life.

## Protocol

### 1. Check the time and energy
```bash
TZ='America/New_York' date '+%Y-%m-%d %H:%M:%S %Z (%A)'
```
Ask for energy state: Green, Yellow, or Red.

### 2. Read the briefing
Check `briefing/` for today's file. If it exists, summarize it. If not, build a quick one from STATE.md.

### 3. Confirm The Thing
What's the one primary objective? If it's not set, help choose it.

### 4. Surface blockers
Any open loops from last shift? Anything urgent in the capture log?

### 5. Begin
State The Thing, energy, and the first action. Then get out of the way.

---

## Rules
- Don't turn this into a 20-minute conversation
- If energy is red, shrink everything to the minimum
- The user sets the heading. The agent supports execution.
