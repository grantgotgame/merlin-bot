## Session Start — Perception Pass

*Percepts before concepts. Establish contact with reality before reasoning from files.*

### 1. Perceive — Gather Evidence

```bash
TZ='America/New_York' date '+%Y-%m-%d %H:%M:%S %Z (%A)'
```

Check the current time, day, and what's changed since last session.

### 2. Orient — Read State

Read `core/STATE.md`. Treat it as a **hypothesis to verify**, not unquestioned truth.
- Check `Last Active` date
- If **> 2 days** since last active: start gently, acknowledge the gap without judgment
- If ≤ 2 days: proceed normally
- Note any contradictions between files and observed reality

### 3. Assess — Energy State

Ask for energy state if not stated:
- 🟢 GREEN: Full power. Stretch goals.
- 🟡 YELLOW: Low energy. Minimums only. Shrink The Thing to 60 min.
- 🔴 RED: Survival. One tiny action. Permission to stop.

### 4. Choose — Confirm The Thing

Confirm or set The Thing for the day. One primary objective. If it ships, the day wins.
