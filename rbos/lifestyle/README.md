# Lifestyle

Rituals, routines, and life management.

## Subdirectories

- `rituals/` — Morning, evening, weekly protocols

## How This Works

Rituals are structured protocols the agent can walk you through. They should be designed for YOUR brain, not for a neurotypical ideal.

Design principles:
- If a ritual causes dread, redesign it (don't blame yourself)
- Stack onto existing behaviors (don't create new ones)
- Minimum viable version always available
- The system serves you, not the other way around
